Шаг 5. Проверяем, занеслись ли параметры в глобальную конфигурацию

Теперь перейдём в файл глобальной конфигурации и убедимся, что параметры занеслись. Заодно посмотрим, как происходит запись параметров в самом файле. Напомним, что глобальная конфигурация на операционной системе Windows по умолчанию находится на диске C в директории пользователя.

В файле есть два раздела:

  • core — был создан сразу после установки Git;
  • user — появился после того, как мы прописали первую команду с user.name.

Внутри раздела содержатся ключи и значения. В данном случае name — это ключ, а его значение — SineYlo, то есть наш ник. Раздел core мы разберём чуть позже, когда будем исправлять ошибку, связанную с неправильным значением пути до редактора кода Visual Studio Code.
